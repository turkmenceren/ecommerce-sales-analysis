# E-Ticaret Satış Analizi Projesi (SQL + Python + Streamlit)

Bu proje, bir e-ticaret işletmesinin ham satış verilerini kullanarak iş zekası çözümleri üretmek amacıyla geliştirilmiştir. Veri temizleme sürecinden interaktif bir dashboard oluşturma aşamasına kadar tüm süreç uçtan uca yönetilmiştir.

## Kullanılan Teknolojiler
* Python (Veri manipülasyonu ve görselleştirme)
* SQL (SQLite ile veritabanı yönetimi ve karmaşık sorgulamalar)
* Streamlit (İnteraktif web arayüzü)
* Pandas & Plotly (Veri analizi ve grafikler)

## Proje Özellikleri
Proje kapsamında aşağıdaki temel analizler gerçekleştirilmektedir:

* Kategori Bazlı Kârlılık: Hangi kategorilerin en yüksek kâr marjına sahip olduğunun tespiti.
* Bölgesel Satış Performansı: Şehir ve eyalet bazında ciro dağılımları.
* Zarar Analizi: İşletmeye net zarar ettiren ürünlerin SQL üzerinden filtrelenmesi.
* Müşteri Analizi: En yüksek harcama yapan VIP müşterilerin belirlenmesi.

## Nasıl Çalıştırılır?
Bilgisayarınıza kütüphaneleri kurduktan sonra terminale aşağıdaki komutu yazarak dashboard'u başlatabilirsiniz:

streamlit run app.py
